Fast geschafft – Zugangsdaten fehlen

Lege eine .env-Datei neben das Skript (Vorlage: .env.example) und trage deine Spotify-App-Daten ein.

  1. Öffne das Spotify Developer Dashboard → Create app.
  2. Trage als Redirect URI exakt ein: {{ redirect_uri }}
  3. Kopiere Client ID + Client Secret in die .env:
SPOTIPY_CLIENT_ID=deine_client_id SPOTIPY_CLIENT_SECRET=dein_client_secret SPOTIPY_REDIRECT_URI={{ redirect_uri }}

Danach Seite neu laden.
